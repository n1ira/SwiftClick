Please create the following icon files for the SwiftClick extension:

1. icon16.png - 16x16 pixels
2. icon48.png - 48x48 pixels
3. icon128.png - 128x128 pixels

Suggested icon design:
- Simple, flat design with a lightning bolt icon (representing "Swift")
- Primary color should be #3498db (blue)
- Include a small magnifying glass or cursor element to represent inspection functionality
- Clean, modern look suitable for a developer tool

These icons are required for the Chrome Web Store listing and will be displayed in various places in the browser UI.
